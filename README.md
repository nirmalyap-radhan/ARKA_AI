# ARKA AI — AI-Powered Travel and Safety Planner

**ARKA AI** is a premium, AI-driven travel intelligence platform designed for seamless trip planning and safety management for Odisha and beyond. 

---

## 🌟 Key Features

- **Personalized Itinerary Generator**: Create custom travel plans in seconds using GPT-4o.
- **AI Assistant Chat**: Get real-time answers and local insights from your travel companion.
- **AI Vision Scanner**: Analyze and map historical sites or objects using advanced AI vision.
- **VR Historical Explorer**: Experience 3D architectural details and historical guidebooks.
- **Odisha-Deep Cultural Insights**: Specialized information on local festivals, traditions, and Odia-language support.
- **Real-Time Safety & Risk Prediction**: Integrated weather, crime, and health alerts for your destination.
- **Travel Logistics Tracker**: Live traffic status and optimal route suggestions.

---

## 🛠️ Tech Stack

- **Frontend**: React, Vite, TypeScript, Tailwind CSS, Shadcn/UI, Lucide React, Framer Motion.
- **Backend**: FastAPI (Python), OpenAI API, Python-Dotenv, Uvicorn.

---

## 🚀 Getting Started

### 1. Prerequisites
- **Node.js** (v18+)
- **Python** (v3.9+)
- **OpenAI API Key**

### 2. Backend Setup
1. Navigate to the `backend` folder.
2. Create and activate a virtual environment.
3. Install dependencies: `pip install -r requirements.txt`.
4. Create a `.env` file and add `OPENAI_API_KEY`.
5. Run the server: `uvicorn main:app --reload --port 8000`.

### 3. Frontend Setup
1. Navigate to the project root.
2. Install dependencies: `npm install --legacy-peer-deps`.
3. Run the development server: `npm run dev`.
